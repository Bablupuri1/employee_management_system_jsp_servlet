# Employee Management System

A full-stack Employee Management System developed using Advanced Java technologies. This project follows the MVC (Model-View-Controller) architecture and provides secure employee management functionalities.

---

## Technologies Used

- Java
- JSP
- Servlet
- JDBC
- MySQL
- MVC Architecture
- HTML
- CSS
- Bootstrap
- Apache Tomcat
- Git & GitHub

---

## Features

- Admin Login
- Add Employee
- View All Employees
- Update Employee Details
- Delete Employee
- Search Employee
- Session Management
- Form Validation
- Database Connectivity using JDBC
- MVC Architecture

## How to Run

1. Clone the repository

```
git clone https://github.com/Bablupuri1/employee_management_system_jsp_servlet.git
```

2. Import project into Eclipse.

3. Configure Apache Tomcat.

4. Create MySQL database.

5. Update database username and password.

6. Run the project on Tomcat.
